PingClaw 便携版
================

1. 务必双击 `Start PingClaw.command`（Mac）或 `Start PingClaw.bat`（Windows）启动。
   不要直接双击 PingClaw.app，否则可能仍读取本机 ~/.openclaw。
2. 所有配置与聊天记录保存在本文件夹的 data/ 目录中。
3. 请勿在运行中拔出 U 盘。
4. 建议使用 USB 3.0 及以上 U 盘。U 盘格式推荐 APFS 或 Mac OS 扩展（日志式）；
   exFAT 下 Python 运行时会自动缓存到本机磁盘（仅运行时，配置仍在 U 盘）。

首次在 Mac 上若无法打开：
  右键 PingClaw.app 或启动脚本 → 打开
